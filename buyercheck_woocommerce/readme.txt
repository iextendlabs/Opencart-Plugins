BuyerCheck is a plugin that integrates with WooCommerce to manage order processing and enhance security.

== Description ==

BuyerCheck is a powerful plugin designed to integrate seamlessly with WooCommerce, providing enhanced order processing capabilities and security features. It allows store owners to set up custom webhooks, manage API keys, and configure actions for risky clients.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/buyercheck` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Use the BuyerCheck settings page to configure the plugin.

== Frequently Asked Questions ==

= How do I get an API key? =
Visit [BuyerCheck](https://buyercheck.bg) to register and obtain your API key.

= What does the plugin do if a client is marked as very risky? =
You can configure the plugin to either do nothing or disable Cash on Delivery (COD) for risky clients.
